A web-based application designed to facilitate safe, transparent, and user-friendly online elections.
The backend is built using Spring Boot, ensuring robust security, scalability, and efficient data handling.
The frontend is developed in Angular, offering a responsive and interactive user interface for voters and administrators.

Key Features:

User Authentication & Authorization – Secure login for voters and election admins.

Encrypted Voting Process – Votes are securely recorded to prevent tampering.

Real-time Results – Instant tallying and display of votes.

Role-based Access Control – Admins can create elections, manage candidates, and monitor results.

Scalable Architecture – Designed to handle a large number of concurrent users.

Tech Stack:

Backend: Spring Boot (Java)

Frontend: Angular

Database: MySQL

Security: JWT-based authentication, HTTPS, data encryption
